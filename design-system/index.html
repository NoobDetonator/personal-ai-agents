NoobDS

Noob Detonator Design System

Sistema neumórfico profissional — tokens, fundações, componentes, motion e acessibilidade.

Tátil por Padrão

Cada elemento comunica profundidade. Sombras outset/inset respondem a interação. Nada é plano sem propósito.

Movimento com Propósito

Nenhuma animação existe só para enfeitar. Motion orienta atenção, confirma ações e dá feedback sensorial.

Acessível por Padrão

Contraste WCAG AA, focus-visible em todos os interativos, reduced-motion embutido. Acessibilidade não é exceção.

Consistência Sensorial

Mesmas curvas de easing, mesmos timings, mesma voz visual em cada componente. O sistema soa como uma peça só.

Do's & Don'ts

DO: Use Primary para a ação principal
Apenas uma ação primary por contexto. Demais ações devem ser Default ou Ghost.
DON'T: Múltiplos Primary lado a lado
Cria confusão hierárquica. Use variantes diferentes para ações secundárias.

Accordion

O sistema opera com agentes hierárquicos. O Principal delega tarefas a Managers, que por sua vez coordenam Workers especializados para execução.
Suportamos Gemini 2.5 Pro, Gemini 2.5 Flash, GPT-4o e Claude 3.5. Cada agente pode ser configurado com um modelo diferente.
Na página de Settings, ative o toggle "Auto-approve". Isso permite que agentes executem ações sem solicitar confirmação manual.
